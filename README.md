# Car Insurance Claim Prediction and Analysis
ML project to predict whether a car insurance claim will occur and estimate its cost. 

Used: Python, Pandas, Scikit-Learn, XGBoost, CatBoost, Matplotlib

– Built a two-step ML pipeline to predict occurrence and cost of car insurance claims using dataset of 10,000+ policyholders.

– Performed extensive data cleaning and preprocessing.

– Built a data pipeline using Scikit-learn to handle scaling, encoding, and missing values.

– Trained classification and regression models like Random Forest, XGBoost, CatBoost, and and Support Vector Machines. Tuned model settings using GridSearchCV and RandomizedSearchCV to improve performance.

– Evaluated performance using F1-score, confusion matrix, and RMSE.
